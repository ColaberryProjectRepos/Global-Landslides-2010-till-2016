Create Map

Ø        Open Visualization pane

Ø        Select Filled Map visual

Ø        Drag the visual to the dashboard and chose Columns Landslide ,Country Name

Ø        From title change to Severity By Location

Ø        From Format Visual, Visual ,Fill color


Maximum add color #de6a73 (Red)

Medium add color #e8d166 (Yellow)

Minimum  add color #118DFF (Blue)


Ø          General: H - Position = 640, V - Position = 188, Width = 640 and Height = 274 



Ø From Robin select Insert then Image chose snippet of the color code to the chart area and resize. Click on text box on the Home pane the type 'low’, 'High' then drag to the chart.
