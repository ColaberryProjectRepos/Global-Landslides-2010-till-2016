Adding Project Title 

Ø  Click on the Text box from Home Menu

Ø  Begin typing the name of the project (Global Landslide from 2010 till 2016)

Ø  Turn the background to “off” position.

Ø  Select the text then change the font size to 32 and font type to  Arial and choose center for Alignment

Ø  select the text then font color to #000000 .

Ø  General:  V - Position = 235, H - Position = 2, Width = 812 and Height = 82
