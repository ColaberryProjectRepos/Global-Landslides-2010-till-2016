Load Excel file into Power BI desktop 6

     (((that how look like Dashboard))) 






Steps

    

    To start Build project follow Instructions :-
Download the Global Landslides Excel file by clicking  icon above and unzip to your folder

Ø   Open Power BI Desktop

Ø   Click Get data on Power BI Desktop

Ø   Under Get Data select Excel workbook

Ø   Select the excel file "Global Landslides" from your folder and open it

Ø   Click on the load button to load the data

Ø  Click on the Data View To see imported data
